Plugin Payment Gateway TriPay Untuk Easy Digital Download

== Description ==

TriPay Payment adalah payment gateway indonesia yang menyediakan beragam metode pembayaran seperti virtual account, convenience store, e-wallet, dll.

Beberapa channel yang tersedia diantaranya :
* BRI VA
* BNI VA
* Mandiri VA
* Maybank VA
* Permata VA
* Indomaret
* Alfamart
* QRIS
* OVO
* LinkAja
* dan terus bertambah.

Silahkan melakukan pendaftaran terlebih dahulu di (https://payment.tripay.co.id) untuk bisa menggunakan plugin ini.

== Installation ==

1. Pastikan Anda telah menginstall plugin Easy Digital Download karena ini merupakan addon untuk Easy Digital Download.
2. Unggah plugin ini ke folder `/wp-content/plugins/edd-tripay`, atau install langsung melalui WordPress plugin secara instan.
3. Aktifkan di menu 'Plugins' WordPress Anda.
4. Masuk ke menu Downloads -> Payment Gateways -> TriPay lalu isi kolom yang tersedia
5. Salin "Callback URL" berupa link contoh: `http://site.com/index.php?edd-listener=tripay` lalu masukkan ke menu Merchant di akun TriPay Payment Anda
6. Lalu pada bagian Metode Pembayaran silahkan pilih metode pembayaran yang ingin digunakan
